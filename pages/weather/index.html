    查询天气信息

    # 接口信息

    • 接口状态 ： 正常
    • 请求方式 ：GET
    • 返回格式 ：JSON
    • 扣除积分数 ：5

    # 请求地址

    http://vamapi.cloud/weather
    1

    # 请求参数

    参数名 必选 类型 描述
    adcode 是 String 城市编码，参照 (opens new window)

    # 响应参数

    参数名称 类型 描述
    code int 响应码
    data.province string 省份名
    data.city string 城市名
    data.weather string 天气现象
    data.temperature string 实时气温，单位：摄氏度
    data.winddirection string 风向描述
    data.windpower string 风力级别，单位：级
    message string 响应描述
